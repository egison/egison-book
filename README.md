# Egison Book

A book introducing the Egison programming language, available in Japanese and English.

## Structure

- `ja/` — Japanese version (日本語版)
- `en/` — English version

## Building

### Japanese version

```bash
cd ja
make
```

### English version

```bash
cd en
make
```

Both require `platex`, `pbibtex`, `dvipdfmx`.
