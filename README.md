# Egison Book

A book introducing the Egison programming language, available in Japanese and English.

## Structure

- `ja/` — Japanese version (日本語版)
- `en/` — English version

## Building

### Japanese version

```bash
cd ja
make
```

Requires `platex`, `pbibtex`, `dvipdfmx`.

### English version

```bash
cd en
make
```

Requires `pdflatex`, `bibtex`.
